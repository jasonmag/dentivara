### Package Explanation

**Starter MVP - PHP 180,000.00**

This package is for a controlled first version, pilot, or proof-of-concept deployment. It is intended to validate the clinic workflow with basic patient, appointment, and billing functions.

Included:
- Basic patient records
- Basic appointment scheduling
- Basic services and pricing setup
- Basic invoice and payment recording
- Admin login and user access
- Single deployment assistance

Limitations:
- Minimal UI polish
- Limited QA/testing coverage
- Limited documentation
- Short support window
- No advanced queue workflow, SMS production integration, compliance dashboard, advanced reporting, insurance claims, or patient portal expansion

**Clinic Operations - PHP 450,000.00**

This package is for a clinic that wants a more usable day-to-day operations system. It includes the main workflows needed by reception, billing staff, and clinic administrators.

Included:
- Everything in Starter MVP
- Improved billing workflow
- Partial payment tracking
- Payment history
- Payment proof upload
- PDF invoices and receipts
- Reception queue board
- Clinic settings
- Basic audit logs
- Basic handover documentation

Limitations:
- Suitable for small to mid-size clinic operations
- Does not include full compliance dashboard expansion, advanced analytics, insurance claims, online payments, or multi-branch architecture

**Full Production Platform - PHP 900,000.00**

This package is for production deployment with stronger accountability. The price includes not only the software features, but also deployment responsibility, QA cycles, documentation, go-live support, and post-go-live warranty.

Included:
- Everything in Clinic Operations
- Compliance dashboard and audit visibility
- Stronger RBAC/security hardening
- Patient portal baseline
- Advanced scheduling rules
- Queue ETA configuration
- Deployment hardening
- Critical workflow testing
- Documentation and handover
- Go-live support
- 30-day post-go-live bug-fix warranty

This package is recommended when the client expects the system to be used as a live clinic operations platform rather than only a pilot or demo.

### Package Difference Summary

The lower packages reduce delivery responsibility, QA depth, documentation, and production hardening. The PHP 900,000.00 package is not priced only as "code"; it is priced as a full production delivery including implementation, testing, deployment, handover, and support accountability.
